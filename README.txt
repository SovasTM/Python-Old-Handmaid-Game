Old Maid is a classic children's card game. 
It can be played by two or more players of almost any age using a standard deck of cards or one of the many specialized decks available. 
It reinforces matching skills and has them actively participating without the need for much assistance if any.

For information on how to play the card game please go to https://www.classicgamesandpuzzles.com/Old-Maid.html
